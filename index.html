Testing Git and Github

Steps to follow

  1. git init: Initalizes Github repository
  2. git add . or git add "file name": git add . adds all files in current directory to a tracked github branch. git add "file name" adds only the specified file.
  3. git commit -m "add title of commit in quotes" -m "add description in second quotes" (not required)
  4. git push origin master
